## Web Security Methodology

Estoy construyendo una metodología propia para investigación web combinando recon, análisis manual, clasificación de vulnerabilidades y documentación de impacto.

La metodología se enfoca en:

- Reconocer superficie expuesta
- Mapear endpoints y flujos sensibles
- Analizar JavaScript del frontend
- Revisar autenticación, sesiones y control de acceso
- Probar manualmente requests y responses
- Validar impacto real
- Clasificar vulnerabilidades con OWASP, CWE y CVE
- Documentar hallazgos de forma clara y reproducible
